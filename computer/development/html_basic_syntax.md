## [Html 基础语法](./html_basic_syntax)

##### *Made by* [AImator.com]()

本章主要介绍了 Html5 的基础语法，代码内容除非特别说明，否则区分大小写。

### DOCTYPE

```html
<!DOCTYPE html>
```

上述语法不区分大小写。

### 字符编码

```html
<meta charset="UTF-8" />
```

上述语法不区分大小写。

### script 标签

```html
<script src="scriptfile.js"></script>
```

### link 标签

```html
<link rel="stylesheet" href="styfile.css">
```
