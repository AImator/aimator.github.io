## [Markdown](./markdown)

##### *Made by* [AImator.com]()

**Markdown** 是轻量级的**文本标记语言**，通过简单易学的标记语法，使普通文本内容具有一定的格式，旨在实现易读易写，成为一种适用于网络的书写语言。

### 学习目标

- 掌握 Markdown 基础语法规则及对应的文字表现形式；
- 熟悉 Markdown 扩展语法，包括制表、清单等高级表现形式；
- 了解 Markdown 的一些高级用法。
